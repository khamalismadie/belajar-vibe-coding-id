# ⚖️ Example: Legal Document Vibe-Checker

Contoh ini menunjukkan penggunaan Vibe Coding untuk membangun alat utilitas yang memanfaatkan kekuatan pemrosesan bahasa alami AI.

## 1. The Vibe (Prompts)

### Prompt 1: File Upload
> "Buatkan dashboard Next.js sederhana yang memiliki area dropzone untuk upload file PDF atau teks. Saya ingin tampilannya terlihat profesional, gunakan warna biru navy dan putih."

### Prompt 2: AI Analysis Logic
> "Integrasikan aplikasi ini dengan API Gemini/Claude. Tugas AI adalah membaca isi dokumen yang di-upload dan mencari 'red flags' atau poin-poin yang berisiko bagi pemilik bisnis (misal: denda tersembunyi, masa kontrak yang terlalu lama, atau pasal non-kompetisi yang tidak adil)."

### Prompt 3: Result Display
> "Tampilkan hasil analisis AI dalam bentuk list yang diberi kategori: 🔴 High Risk, 🟡 Warning, dan 🟢 Safe. Tambahkan tombol untuk mendownload ringkasan analisis dalam format Markdown."

## 2. Real-World Use Case
Sangat berguna bagi freelancer atau UMKM untuk mengecek kontrak cepat sebelum dikirim ke pengacara.
