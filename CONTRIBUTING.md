# Panduan Kontribusi

Kami senang Anda ingin berkontribusi di **belajar-vibe-coding-id**! Berikut adalah cara Anda bisa ikut serta:

## Cara Berkontribusi

### 1. Menambahkan Prompt Baru
Jika Anda punya prompt "sakti" yang sering Anda gunakan:
1. Buat file baru di folder `/prompts/`.
2. Gunakan format Markdown.
3. Berikan nama file yang deskriptif (misal: `debug-error-react.md`).

### 2. Memperbaiki Dokumentasi
Jika ada istilah yang kurang tepat atau panduan yang sulit dipahami:
1. Edit file di folder `/docs/`.
2. Pastikan menggunakan Bahasa Indonesia yang mudah dipahami oleh developer pemula.

### 3. Menambahkan Contoh Proyek
Punya proyek kecil hasil *vibe coding*?
1. Tambahkan deskripsi proyek di folder `/examples/`.
2. Lampirkan link repositori atau kode sumbernya.
